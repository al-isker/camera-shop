## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Libraries used
- Next 14
- React 18,
- TanStack Query v5,
- Axios,
- Material UI
- Framer Motion
